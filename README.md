# График освещенности

Веб-приложение для расчета графика освещенности по координатам.
Работает полностью офлайн после первой загрузки.

## Возможности

- Расчет времени рассвета, заката, сумерек
- Определение полярного дня и полярной ночи
- Визуальный график освещенности
- Работает без интернета
- Устанавливается как приложение на iPhone

## Установка на Netlify через GitHub (с iPhone)

### Шаг 1: Создание репозитория на GitHub

1. Откройте Safari на iPhone
2. Перейдите на https://github.com
3. Войдите в аккаунт (или зарегистрируйтесь)
4. Нажмите кнопку **+** (вверху справа) → **New repository**
5. Введите название: `light-calculator` (или любое другое)
6. Выберите **Public**
7. Нажмите **Create repository**

### Шаг 2: Добавление файлов

**Файл 1: index.html**
1. В репозитории нажмите **Add file** → **Create new file**
2. В поле имени файла введите: `index.html`
3. Скопируйте содержимое файла index.html (из этого проекта)
4. Вставьте в большое текстовое поле
5. Прокрутите вниз → нажмите **Commit changes** → **Commit changes**

**Файл 2: manifest.json**
1. Снова **Add file** → **Create new file**
2. Имя файла: `manifest.json`
3. Вставьте содержимое manifest.json
4. **Commit changes**

**Файл 3: sw.js**
1. **Add file** → **Create new file**
2. Имя файла: `sw.js`
3. Вставьте содержимое sw.js
4. **Commit changes**

### Шаг 3: Подключение Netlify

1. Откройте https://app.netlify.com в Safari
2. Нажмите **Sign up** (или **Log in**)
3. Выберите **Sign up with GitHub** → разрешите доступ
4. Нажмите **Add new site** → **Import an existing project**
5. Выберите **Deploy with GitHub**
6. Найдите ваш репозиторий `light-calculator` → нажмите на него
7. Настройки:
   - **Branch to deploy**: main (или master)
   - **Build command**: оставьте пустым
   - **Publish directory**: оставьте пустым
8. Нажмите **Deploy site**

### Шаг 4: Получение ссылки

1. Подождите 1-2 минуты, пока идет деплой
2. Когда появится зеленая галочка, скопируйте ссылку вида:
   `https://random-name-12345.netlify.app`
3. Откройте эту ссылку в Safari

### Шаг 5: Установка на iPhone

1. Откройте сайт в Safari
2. Нажмите кнопку **Поделиться** (квадрат со стрелкой вверх)
3. Прокрутите вниз → выберите **На экран "Домой"**
4. Нажмите **Добавить**
5. Готово! Теперь приложение на главном экране

## Работа без интернета

После первого открытия приложение сохраняется в кэш браузера и работает полностью офлайн.
Все вычисления производятся локально в браузере.

## Как пользоваться

1. Введите широту и долготу (или используйте значения по умолчанию)
2. Выберите период (начальную и конечную дату)
3. Нажмите **Построить график**
4. Изучите график освещенности

### Примеры координат

- Санкт-Петербург: 59.93, 30.34
- Мурманск: 68.97, 33.07
- Москва: 55.75, 37.62
- Северный полюс: 90.00, 0.00
